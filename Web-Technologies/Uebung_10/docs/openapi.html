API Documentation

OpenAPI 3.0 Specification for the Meta Playlists API

📋 Overview

Base URL http://localhost:5001/api
Format JSON
Authentication None (prototype)
Version 1.0.0

📦 Data Models

Track

{
  "title": "string",      // Track name (required)
  "link": "string",       // URL to streaming service (required)
  "duration": "string"    // Format: "mm:ss" e.g. "3:45" (required)
}

Playlist

{
  "PlaylistName": [       // Key is the playlist name
    { Track },            // Array of Track objects
    { Track },
    ...
  ]
}

🔌 Endpoints

GET /playlists

Get all playlists

Returns a dictionary of all playlists with their tracks.

Response

200 OK 
{
  "Rock Classics": [
    {"title": "Bohemian Rhapsody", "link": "...", "duration": "5:55"}
  ],
  "Chill Vibes": []
}

Try it

curl http://localhost:5001/api/playlists
POST /playlists

Create a new playlist

Request Body

{
  "name": "My New Playlist"
}

Responses

201 Created Playlist created successfully

{"message": "Playlist 'My New Playlist' created", "name": "My New Playlist"}

400 Bad Request Missing name field

409 Conflict Playlist already exists

Try it

curl -X POST http://localhost:5001/api/playlists \
  -H "Content-Type: application/json" \
  -d '{"name": "My New Playlist"}'
GET /playlists/{name}

Get a specific playlist

Path Parameters

Parameter Type Description
name string Playlist name (URL encoded)

Responses

200 OK

[
  {"title": "Track 1", "link": "...", "duration": "3:20"},
  {"title": "Track 2", "link": "...", "duration": "4:15"}
]

404 Not Found Playlist doesn't exist

Try it

curl http://localhost:5001/api/playlists/Rock%20Classics
PUT /playlists/{name}

Update playlist tracks

Replaces all tracks in the playlist with the provided array.

Request Body

[
  {"title": "New Track", "link": "https://...", "duration": "3:00"},
  {"title": "Another", "link": "https://...", "duration": "4:30"}
]

Responses

200 OK

{"message": "Playlist 'Rock Classics' updated", "tracks": [...]}

404 Not Found Playlist doesn't exist

Try it

curl -X PUT http://localhost:5001/api/playlists/Rock%20Classics \
  -H "Content-Type: application/json" \
  -d '[{"title": "Highway to Hell", "link": "...", "duration": "3:28"}]'
DELETE /playlists/{name}

Delete a playlist

Responses

200 OK

{"message": "Playlist 'Rock Classics' deleted"}

404 Not Found Playlist doesn't exist

Try it

curl -X DELETE http://localhost:5001/api/playlists/Rock%20Classics
