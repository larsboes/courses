Kustomize

Kustomize = Kubernetes + Customize. Es ermoeglicht umgebungsspezifische Konfiguration ohne Duplikation.

┌─────────────────────────────────────────────────────────────────┐
│  KUSTOMIZE STRUKTUR                                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  ┌──────────────────────────────────────────────────────┐       │
│  │                      base/                            │       │
│  │              (gemeinsame Ressourcen)                  │       │
│  └────────────────────────┬─────────────────────────────┘       │
│                           │                                      │
│          ┌────────────────┼────────────────┐                    │
│          ▼                ▼                ▼                    │
│  ┌──────────────┐ ┌──────────────┐ ┌──────────────┐            │
│  │ overlays/    │ │ overlays/    │ │ overlays/    │            │
│  │ uebung-11/   │ │ uebung-12/   │ │ production/  │            │
│  │              │ │              │ │              │            │
│  │ + nginx      │ │ + flask      │ │ + replicas:5 │            │
│  │              │ │ + couchdb    │ │ + secrets    │            │
│  └──────────────┘ └──────────────┘ └──────────────┘            │
│                                                                  │
│  Befehl: kubectl apply -k overlays/uebung-12/                   │
│          └─ -k = Kustomize mode                                 │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘

Base

YAML base/kustomization.yaml 
apiVersion: kustomize.config.k8s.io/v1beta1
kind: Kustomization

namespace: fhdw-wta              # ◄── Default Namespace

resources:                       # ◄── Welche Dateien
  - namespace.yaml

commonLabels:                    # ◄── Labels fuer ALLE Ressourcen
  course: web-technologies
  institution: fhdw

Overlay

YAML overlays/uebung-12/kustomization.yaml 
apiVersion: kustomize.config.k8s.io/v1beta1
kind: Kustomization

namespace: fhdw-wta

resources:                       # ◄── Erbt von base + eigene Dateien
  - ../../base
  - couchdb-pvc.yaml
  - couchdb-secret.yaml
  - couchdb-deployment.yaml
  - couchdb-service.yaml
  - flask-configmap.yaml
  - flask-deployment.yaml
  - flask-service.yaml

commonLabels:                    # ◄── Zusaetzliche Labels
  exercise: uebung-12
  app: meta-playlist

1. Namespace setzen

YAML 
namespace: fhdw-wta    # Alle Ressourcen bekommen diesen Namespace

2. Labels hinzufuegen

YAML 
commonLabels:
  app: meta-playlist   # Wird zu allen Ressourcen hinzugefuegt

3. Prefixes/Suffixes

YAML 
namePrefix: prod-     # Alle Namen werden: prod-flask-webserver
nameSuffix: -v2       # Alle Namen werden: flask-webserver-v2

Bash 
# Kustomize Ergebnis anzeigen (ohne anzuwenden)
kubectl kustomize overlays/uebung-12/

# Mit Kustomize deployen
kubectl apply -k overlays/uebung-12/

# Mit Kustomize loeschen
kubectl delete -k overlays/uebung-12/
