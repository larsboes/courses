Detaillierte Beschreibungen

  1. Installieren Sie Docker
  2. Legen Sie eine Datei docker-compose.yml an
  3. Starten Sie den Container mit docker compose up -d
  4. Schauen Sie sich das Netzwerk an: docker network ls und docker network inspect
  5. Starten Sie eine Shell: docker exec -it container1 /bin/bash
  6. Prozesse pruefen: ps und ps -ef
  7. Namespaces anschauen: lsns
  8. Netzwerk testen: ifconfig, ping
  9. Container beenden: docker compose down

from flask import Flask
webserver = Flask(__name__)

@webserver.route('/')
def home():
    return '<h1>Willkommen auf der minimalen WTA-Homepage!</h1>'

if __name__ == '__main__':
    webserver.run(host='0.0.0.0', port=8000)
